Visão Geral

Equação quadrática, ou equação de segundo grau, é uma equação polinomial de grau 2. É dada pela seguinte fórmula geral:

ax2 + bx + c = 0
Onde:
x é uma variável.
a, b e c são constantes.

A resolução desta equação é dada pela descoberta do(s) valor(s) de x para determinados valores das constantes a, b, c.
