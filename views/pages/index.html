Descrição

Equação quadrática, ou equação de segundo grau, é uma equação polinomial de grau 2. É data pela seguinte fórmula geral:

ax2 + bx + c = 0
Onde:
x é uma variável.
a, b e c são constantes.

A resolução desta equação se dá pela descoberta do valor(s) de x dados valores das constantes a, b, c.
